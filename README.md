AI Resume Analyzer

Overview
An AI-powered Resume Analyzer built using Python, Flask, and NLP techniques. The application analyzes uploaded resumes, extracts skills, and provides ATS-style feedback and recommendations.

Features
- Resume Upload
- Skill Extraction
- ATS-style Feedback
- Skill Recommendations

 Technologies Used
- Python
- Flask
- HTML
- CSS
- Pandas
- NLP

Project Structure
AI-Resume-Analyzer/
├── app.py
├── Requirements.txt
├── skills.csv
├── templates/
├── static/
└── uploads/

How to Run
pip install -r Requirements.txt

python app.py
